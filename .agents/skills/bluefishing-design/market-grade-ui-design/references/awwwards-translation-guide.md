# Awwwards Translation Guide

Awwwards can improve taste, composition and motion, but Bluefishing is an operations product. Translate inspiration carefully.

## Use From Awwwards

- Strong visual hierarchy.
- Intentional negative space in page headers.
- Product screenshots with real UI.
- Smooth section reveals on marketing pages.
- Typography rhythm.
- Memorable first viewport for public pages.

## Avoid From Awwwards

- Hidden navigation.
- Scroll hijacking.
- Low contrast text.
- Huge type inside dashboards.
- Decorative 3D or blobs without product meaning.
- Slow loading hero effects.
- Mystery interactions.

## Dashboard Translation

Instead of editorial animation, use:

- Sticky table headers.
- Drawer transitions.
- Status transitions.
- Skeleton states.
- Command menu reveal.
- Timeline completion.

## Marketing Translation

Marketing can be more expressive:

- First viewport should show Bluefishing name and product screenshot.
- Use a dark operational command-center feel.
- Reveal one workflow visually: email -> invoice -> action -> DIN -> archive.
- Keep claims specific: fewer manual steps, fewer missed payments, clearer customs status.
